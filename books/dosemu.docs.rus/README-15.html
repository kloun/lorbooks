15. Команды и утилиты

Вот некоторые программы, которые облегчат вам использование Dosemu.

15.1 Программы

bootoff.com

отключает загрузочный файл для доступа к диску, см. examples/config.dist в разделе bootdisk

booton.com

включает доступ к загрузочному файлу см. examples/config.dist в разделе bootdisk

chdir.com

изменяет рабочий каталог Unix для Dosemu (при помощи chdir(2))

dosdbg.com

изменяет настройки отладчика из Dosemu

  • dosdbg -- показывает текущее состояние
  • dosdbg <string>
  • dosdbg help -- показывает инструкцию

dumpconf.exe

???

eject.com

извлекает CD-ROM из дисковода

emumouse.com

тонкая подстройка встроенного в Dosemu драйвера мыши.

  • emumouse h -- показывает инструкцию

exitemu.com

завершение Dosemu

getcwd.com

выдает текущий каталог Unix для Dosemu (с помощью getcwd(2))

isemu.com

определяет версию Dosemu и возвращает значение больше 0 при работе в Dosemu

lancheck.exe

???

lredir.com

подставляет каталог Linux в Dosemu

  • lredir -- показывает текущие подстановки
  • lredir D: LINUX\FS\tmp -- подставляет /tmp в качестве диска D:
  • lredir help -- показывает инструкцию

mgarrot.com

отдает циклы cpu, когда Dosemu простаивает

speed.com

задает использование CPU (HogThreshold) из Dosemu

system.com

интерфейс к вызову system(2)...

unix.com

выполняет Unix-команду из Dosemu

  • unix -- показывает инструкцию
  • unix ls -al -- показывает содержимое текущего каталога Linux
  • осторожно! наберите "unix" и прочтите инструкцию

cmdline.exe

Читает /proc/self/cmdline и помещает строки наподобие "var=xxx", найденные в командной строке, в переменные окружения DOS. Пример использования в командной строке запуска dos: 

	dos "autoexec=echo This is a test..."
последующее выполнение 
 
	C:\cmdline < D:\proc\self\cmdline
        %autoexec%
напечатает "This is a test..." в DOS

vgaoff.com

отключает параметр vga

vgaon.com

включает параметр vga

15.2 Драйверы

полезные драйверы для Dosemu

cdrom.sys

позволяет непосредственный доступ к дисководам CD-ROM из Dosemu

ems.sys

включает EMM в Dosemu

emufs.sys

подставляет каталог Unix в Dosemu
