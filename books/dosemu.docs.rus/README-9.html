9. Регистрация действий DOSEmu

This section was written by Erik Mouw <J.A.K.Mouw@et.tudelft.nl> on February 17, 1996 and last edited on 21 June 1997.

Этот раздел описывает возможности регистрации в DOSEMU.

DOSEMU может регистрировать все действия через email или syslogd(8). Если вы хотите включить регистрацию, создайте файл /etc/dosemu.loglevel. В нем вы можете установить уровень регистрации как почтовой, так и через syslog в очень простой форме. Распознаются только четыре ключевых слова:

mail_error

посылать сообщения об ошибках пользователю root

mail_always

посылать сообщения о любом использовании DOSEMU пользователю root

syslog_error

записывать сообщения об ошибках через syslogd(8)

syslog_always

записывать все использование DOSEMU через syslogd(8)

Строки, начинающиеся с '#' - комментарии.

примеры:

#
# отсылать все root-у, но не записывать в журнал
#
mail_always

#
# сообщения об ошибках отылать root-у, записывать все
# (рекомендуемый способ)
#
mail_error
syslog_always

#
# заносить в журнал только сообщения об ошибках
#
syslog_errors
