3. Основы графического интерфейса

Этот подраздел коротко описывает пользовательский интерфейс Редактора пиктограмм.

Окно Редактора пиктограмм разделено на 4 части: основная панель инструментов, панель инструментов редактирования, координатная сетка и панель отображения пиктограммы.

3.1 Основная панель инструментов

New File

Создать новую пиктограмму. Если текущий рабочий файл был изменен -- вам предложат сохранить изменения. Затем откроется диалог о том, каким образом требуется создать пиктограмму: рисовать самостоятельно или использовать шаблон (работа с шаблонами требует наличия kdelibs > 1.0).

Open

Открыть существующий файл с пиктограммой.

Save

Сохранить текущую пиктограмму в файл.

Print

Распечатать пиктограмму.

Cut

Вырезать пиктограмму целиком и поместить ее в буфер обмена.

Copy

Скопировать всю пиктограмму в буфер обмена.

Paste

Вставить содержимое буфера обмена как новое изображение (если буфер обмена содержит корректные данные пиктограммы).

Resize

Изменить размер пиктограммы [ширина X высота] (требуется kdelibs > 1.0)

Gray scale

Преобразовать цветное изображение пиктограммы в шкалу градаций серого. При этом в изображении могут появиться цвета, не соответствующие палитре пиктограмм KDE.

Zoom in

Увеличить.

Zoom out

Уменьшить.

Zoom factor

Увеличить до определенного значения.

Toggle grid

Включить/выключить отображение координатной сетки.

New Icon Editor

Открыть новое окно Редактора пиктограмм.

3.2 Панель инструментов редактирования пиктограмм

Эта панель содержит инструменты для работы с пиктограммами.

Find color

Найти цвет. Не изменяет пиктограммы, но меняет текущий цвет рисования на указанный пользователем.

Freehand

Свободное рисование от руки.

Select

Выбрать (отметить) участок пиктограммы.

Line

Рисовать линию.

Rectangle

Рисовать прямоугольник.

Filled rectangle

Рисовать заполненный прямоугольник.

Circle

Рисовать окружность.

Filled circle

Рисовать заполненную окружность.

Ellipse

Рисовать эллипс.

Filled ellipse

Рисовать заполненный эллипс.

Spray

Рисовать разбрызгивателем.

Flood fill

Заполнить область текущим цветом..

Eraser

Ластик. Рисовать прозрачным (невидимым цветом).

3.3 Координатная сетка

Координатная сеть представляет собой область, где размещается изображение пиктограммы.
