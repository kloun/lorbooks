4. Описание пунктов меню.

4.1 File

В меню работы с файлами доступны следующие функции:

New Window

Открыть новое окно Редактора пиктограмм.

New ...

Создать новую пиктограмму используя шаблон или указав размеры изображения.

Open

Открыть существующий файл с пиктограммой.

Save

Сохранить текущую открытую пиктограмму.

Save as

Сохранить текущую пиктограмму под другим именем.

Print

Напечатать пиктограмму.

Close

Закрыть окно Редактора пиктограмм.

Exit

Закрыть все открытые окна Редактора пиктограмм.

4.2 Edit

Меню Редактирования (Edit) содержит следующие пункты:

Cut

Вырезать пиктограмму целиком и поместить ее в буфер обмена.

Copy

Скопировать всю пиктограмму в буфер обмена.

Paste

Вставить содержимое буфера обмена как новое изображение (если буфер обмена содержит корректные данные пиктограммы).

Clear

Очистить координатную сетку и заполнить ее прозрачным цветом.

Select all

Выделить всю пиктограмму.

4.3 View

Zoom in

Увеличить.

Zoom out

Уменьшить.

Zoom factor

Увеличить до определенного значения.

4.4 Tools

Freehand

Свободное рисование от руки.

Select

Выделить (отметить) часть пиктограммы.

Line

Рисовать линию.

Rectangle

Рисовать прямоугольник.

Filled rectangle

Рисовать заполненный прямоугольник.

Circle

Рисовать окружность.

Filled circle

Рисовать заполненную окружность.

Ellipse

Рисовать эллипс.

Filled ellipse

Рисовать заполненный эллипс.

Spray

Рисовать разбрызгивателем.

Flood fill

Заполнить область текущим цветом.

Eraser

Ластик. Рисовать прозрачным цветом (невидимым).

4.5 Options

Toggle grid

Включить/выключить координатную сетку.

4.6 Help

В этом меню вы можете посмотреть информацию о Редакторе пиктограмм.
