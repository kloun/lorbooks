3. Пункты меню

3.1 Файл

Новое окно

Создает новое окно KZip

Новый...

Создает новый архив. Тип архива определяется по расширению, которое вы даете новому файлу.

Открыть...

Открывает архив. Если уже был открыт какой-либо архив, он будет закрыт.

Закрыть

Закрывает текущее окно KZip. Если открытых окон больше не осталось, программа завершает работу.

Выход

Завершение работы KZip.

3.2 Редактирование

Распаковать

Разархивирует помеченные в списке файлы и каталоги.

Просмотреть файл

Вызывает диалог, запрашивающий имя программы, с помощью которой вы хотите просмотреть указанный файл.

Удалить файл

Удаляет выбранный файл из архива. Эта возможность доступна не для всех форматов архивов.

3.3 Настройки

Установить подкаталог архивации...

Вызывает диалоговое окно, запрашивающее каталог, где вы храните свои архивы. После установки каталога нажатие пиктограммы "home" будет отображать список архивов в этом каталоге.

Параметры добавления файлов...

Вызывает диалог для установки двух параметров. Если вы хотите, чтобы файлы заносились в архив с указанием полного пути, отметьте это. Иначе полный путь не будет сохраняться в архиве.

Вторая возможность должна быть включена в том случае, если вы хотите добавлять к архиву только более новые файлы. Для некоторых архивных форматов это может не работать.

3.4 Помощь

Помощь

Вызывает справку по программе KZip.

О программе...

Вызывает общую информацию о KZip.
