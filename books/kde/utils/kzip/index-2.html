2. Основы графического интерфейса

2.1 Установка каталога архивов

KZip будет автоматически запоминать ваш каталог с архивами. Вы должны вначале установить его из пункта меню "Установить подкаталог архивации" из меню "Настройки". Введите полный путь, например, /home/rob/dloads. Для быстрого перехода в этот каталог просто щелкните на иконку панели инструментов "home". Будет показан список всех файлов в этом каталоге. Чтобы открыть архив, щелкните мышью на нужный файл.

2.2 Открытие и просмотр архивов

Существует несколько способов открыть архив. Первый -- это нажать пиктограмму "открыть" на панели инструментов или выбрать пункт "Открыть" из меню "Файл". Другой способ -- это щелкнуть на пиктограмму архива в вашем каталоге архивов.

2.3 Работа с файлами

Перед тем как вы сможете работать с архивом, его необходимо открыть. Чтобы просмотреть файл из архива, щелкните по нему мышью или выберите пункт "Просмотр" из меню "Редактировать". Появится диалог, в котором необходимо выбрать программу для просмотра файла. Наберите kless, kview или то, что подходит для просмотра этого файла. Выберите "Редактировать...Извлечь" для извлечения файла из архива в указанный каталог. Выберите "Редактировать...Удалить" для удаления файла из архива.

2.4 Разархивирование

Открытый архив можно быстро разархивировать в любой каталог. Для этого щелкните на пиктограмму "Распаковать" или выберите пункт меню "Файл...Распаковать в". Появится диалог, предлагающий выбрать каталог, в который будет производиться разархивирование. Наберите полный путь, например /home/rob/new_kde_stuff. Появится индикатор прогресса выполнения команды.

2.5 Создание архива и добавление к нему файлов

Чтобы создать новый архив, выберите пункт "Новый" из меню "Файл". Файлы могут быть добавлены к открытому архиву перетаскиванием из KFM. За один раз можно добавить целые каталоги или несколько файлов.
