2. Установка

2.1 Где взять

Программа форматирования KFloppy является частью проекта KDE http://www.kde.org. KFloppy можно найти по адресу ftp://ftp.kde.org/pub/kde/, основном ftp-сайте проекта KDE.

2.2 Требования

Для лучшей компиляции KFloppy вам необходимы последние версии KDE библиотек. Все требуемые библиотеки и сам KFloppy Formatter можно найти по адресу ftp://ftp.kde.org/pub/kde/.

2.3 Компиляция и установка

Для лучшей компиляции и установки KFloppy Formatter на вашу систему -- читайте READ.ME в базовом каталоге дистрибутива KFloppy. 

 
% ./configure
% make
% make install

Поскольку KFloppy использует autoconf у вас не должны возникнуть проблем при его компиляции. Если они все-таки возникнут, пожалуйста, сообщите о них в список рассылки KDE.
