3. Основы графического интерфейса

Использование KMix очень просто. Каждому устройству, содержащемуся в вашей звуковой плате соответствует специальный ползунок, регулирующий громкость. Моно-устройствам соотвествует только один ползунок, стерео-устройствам - два. Вы можете объединить 2 ползунка стерео-устройства в один. Внизу окна Kmix расположен дополнительный регулятор баланса.

3.1 Ползунки, регулирующие громкость

Ползунки, регулирующие громкость, показывают уровень громкости выбранного устройства. Верх -- максимальный уровень громкости, низ -- минимальный. Ползунок может быть трёх цветов:

  • Нормальный Все функционирует в штатном режиме.
  • Черный Устройство отключено.
  • Красный В данный момент это устройство - источник записи. Это устройство так же может быть отключено.

Так же у ползунков есть контекстное меню. Возможны 3 пункта меню, но показаны только те, которые можно выбрать.

  • Mute [On,Off] Отключить или включить устройство.
  • (Un-)Split Показать 1 или 2 ползунка. Применимо только для стерео-устройств.
  • Record Source Установить данное устройство как источник записи. Применимо только для устройств, позволяющих вести через них запись.

3.2 Регулятор баланса

С помощью этого ползунка вы можете регулировать распределение громкости между левым и правым каналом. Этот регулятор действует на все стерео-устройства одновременно(моно-устройства игнорируются). По умолчанию ползунок располагается посередине. Перемещение ползунка влево снижает громкость правого громкоговорителя, вправо -- левого. У регулятора баланса есть контекстное меню, которое доступно по щелчку правой кнопки мыши. Вы можете установить ползунок в несколько стандартных положений, используя это меню.

3.3 Несколько звуковых плат.

KMix позволяет работать с несколькими звуковыми платами. Если вы запустили KMix из K панели, то используется эквалайзер по умолчанию. Для Open Sound System это /dev/mixer. Для других операционных систем и драйверов эти значения могут меняться.

Если вы хотите, чтобы KMix использовал специально указанную звуковую плату (первичную, вторичную и т.д.) вы можете указать это при запуске KMix. Для этого вам следует запустить эквалайзер с номером устройства. 0 означает устройство по умолчанию, 1 -- основную звуковую плату, 2 -- вторичную и так далее.

3.4 Привязка к клавишам

KMix разрешает использовать следующие привязки к клавишам.

  • Up Arrow Увеличить громкость на 1%
  • Down Arrow Уменьшить громкость на 1%
  • Page Up Увеличить громкость на 10%
  • Page Down Уменьшить громкость на 10%
  • Control-M Включить или отключить верхнее меню
