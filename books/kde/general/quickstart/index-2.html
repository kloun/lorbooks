2. Обзор KDE

Этот раздел предназначен для пользователей, которые предпочитают обучаться методом проб и ошибок, и которым только для начала нужны краткие ориентиры. Дальнейшие разделы предлагают более тщательное введение в среду, с полезными подсказками и трюками. Если вам не терпится начать, пролистайте этот раздел, поиграйтесь немного, а затем тщательно изучайте следующие разделы руководства по мере необходимости.

Замечание: KDE предоставляет в высокой степени настраиваемую среду рабочего стола. Наш обзор предполагает, что вы пользуетесь настройками по умолчанию.

2.1 Рабочий стол KDE

Типичный рабочий стол KDE состоит из трех частей:

  • Панель в нижней части экрана, используемая для запуска приложений и переключения между рабочими столами. Помимо прочего, она содержит Диспетчер приложений (большую пиктограмму "K"), который предлагает меню приложений, которые стартуют по нажатию кнопки мыши.

  • Панель задач в верхнем левом углу экрана, используемая для переключения между работающими приложениями и управления ими. Щелканье по приложению на этой панели позволяет переключиться на это приложение.

  • Сам рабочий стол на котором могут быть размещены часто используемые файлы и папки. KDE поддерживает одновременно несколько рабочих столов, каждый из которых имеет свои окна. Для переключения между рабочими столами нужно щелкать по пронумерованным кнопкам на панели.

2.2 На старт, внимание, марш!

Вот несколько советов для начала работы.

  • Для запуска приложения щелкните по кнопке "K" на панели (называемой Диспетчером приложений и выберите пункт меню.
  • Если же щелкнуть по пиктограмме -- в виде папки с домиком -- на панели приложений, то вы получите доступ к файлам своего домашнего каталога через kfm, утилиту файл-менеджер KDE (KDE's File Manager).
  • Выберите Диспетчер Приложений > Полезное > Терминал чтобы войти в командную строку Unix, или нажмите Alt-F2 чтобы оказаться в окне малой командной строки (mini command prompt), где можно выполнить одну команду.
  • Для настройки KDE выберите пункт Центр управления KDE в меню Диспетчера приложений KDE.
  • Нажимайте Alt-Tab для переключения между приложениями и Ctrl-Tab для переключения между рабочими столами с помощью клавиатуры.
  • Щелканьем правой кнопки мыши вызываются контекстные меню Панели, Рабочего стола и многих прикладных программ KDE.
