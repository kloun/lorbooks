3. Настройка kdm

Имеется программа с графическим интерфейсом для настройки kdm. Вы найдете ее в меню K/Настройки/Приложения. С помощью этой программы вы можете настроить некоторые параметры kdm (хранящиеся в kdmrc), но учтите, что kdm, как и xdm, требует точной настойки, так что читайте дальше.

Настройки, специфичные для kdm, устанавливаются редактированием файла "$KDEDIR/share/config/kdmrc". В kdmrc есть три раздела:

3.1 KDM

Параметры в этом разделе: 

StdFont
FailFont
GreetFont
GreetString
SessionTypes
LogoPixmap
ShutdownButton
Shutdown
Restart
UserView
Users
NoUsers
SortUsers
GUIStyle

Краткое описание параметров:

*Font

строка описания шрифта следующего формата: Courier,12,5,iso-8859-1,50,0.

GreetString

- заголовок окна входа в систему kdm.

SessionTypes

- разделенный ``;'' список аргументов сеанса. По умолчанию - SessionTypes=kde;failsafe. Приведенный выше пример дополнительно использует менеджеры окон fvwm2 и fvwm95.

LogoPixmap

Измените этот параметр, если вам не нравится логотип.

ShutdownButton, Shutdown and Restart

ShutdownButton определяет, в каких случаях видна кнопка выключения и может принимать значения All, None, RootOnly and ConsoleOnly. Shutdown и Restart - команды, которые следует выполнить для выключения/перезагрузки. По умолчанию - /sbin/halt и /sbin/reboot. All - кнопки выключения/перезагрузки доступны всем, None - никому, RootOnly - только привилегированному пользователю, если вы установите ConsoleOnly, будут невозможны выключение/перезагрузка с удаленного терминала.

UserView

Если этот параметр равен ``1'', окно kdm будет показывать изображение для каждого пользователя.

Users

- разделенный ``;'' список имен пользователей, для которых будут установлены изображения. Если UserView установлен равным 1, а параметра Users нет, kdm ищет пользователей в файле паролей. Выводятся изображения тех пользователей, у которых есть реальное имя, кроме входного имени, домашний каталог и оболочка - остальное в файле паролей игнорируется. Kdm ищет в $KDEDIR/share/apps/kdm/pics/users/ портрет каждого пользователя. Если существует картинка Ваня.xpm, kdm использует ее в качестве портрета пользователя Ваня, иначе kdm использует default.xpm. Так что, если у вас есть маленькое фото каждого из пользователей вашей системы, поместите их в $KDEDIR/share/apps/kdm/pics/users/ для развлечения при входе в систему через kdm ;-) NOTE: Kdm работает с картинками разного размера, но очень маленькие и очень большие картинки рядом могут выглядеть некрасиво.

NoUsers

- разделенный ``;'' список имен пользователей, которых не надо показывать. Используйте этот параметр, чтобы спрятать ``псевдопользователей'' вроде sync и shutdown.

SortUsers

- если этот параметр равен ``0'', список пользователей отображается в том порядке, в котором он был прочитан KDE. По умолчанию список пользователей отображается в алфавитном порядке.

GUIStyle

может быть либо Motif либо Windows.

3.2 KDMDESKTOP

Этот раздел управляет фоном. Параметры: 

BackgroundPicture
BackgroundColor
BackgroundPictureTile
BackgroundPictureCenter
FancyBackground

Первый - имя файла изображения (полный путь), второй - название цвета, остальные - логические значения (0 или 1). Заметьте, что FancyBackground=1 выглядит замечательно, но работает очень медленно.

3.3 Locale

Этот раздел должен содержать только один параметр.

Language

- язык, который будет использовать kdm. Параметр можно изменить внутри KDE. Значения - обычные en/de/fr/da/ru и т.д.
