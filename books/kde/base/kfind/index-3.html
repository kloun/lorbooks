3. Основы графического интерфейса

В этом разделе кратко описан интерфейс пользователя Kfind.

Окно Kfind разделено на три зоны: панель инструментов, диалог параметров поиска и зона просмотра. Зона просмотра и строка состояния становятся видимыми только после получения результатов поиска.

3.1 Панель инструментов

Начать поиск

Начинает новый поиск с параметрами, указанными в зоне диалога.

Новый поиск

Удаляет результаты и начинает поиск с новыми параметрами.

Остановить поиск

Останавливает идущий процесс поиска.

Открыть

Просит Kfm выполнить действие по умолчанию для выбранного файла.

Добавить в архив

Добавляет выделенный файл к выбранному архиву.

Удалить

Удаляет выделенные файлы и просит Kfm перечитать каталог.

Свойства

Показывает свойства выделенного файла.

Открыть папку, в которой содержится файл

Просит Kfm открыть папку, содержащую выделенный файл.

Сохранить результаты поиска

Сохраняет результаты в файл, указанный в диалоге Настройки->Установки.

Помощь

Показывает документацию по KFind.

Выход

Выход из KFind.

3.2 Диалог задания параметров поиска

Этот диалог -- основная часть KFind. Он используется для задания параметров программы поиска. Есть три группы параметров поиска:

Имя Р&асположение

Этот диалог используется для задания имени файла, который надо искать. Имя файла вводится в комбинированный список и может содержать следующие метасимволы:

? любой символ

* любая последовательность символов

[ ] символ из указанного множества, например, [a-zA-Z0-9] означает любую строчную или прописную латинскую букву или цифру.

\ буква, цифра или точка

Комбинированный список хранит прошлые запросы. Кнопка Просмотр используется для указания каталога, в котором будет проведен поиск. Нужно ли просмотреть и подкаталоги, определяется переключателем Рекурсивно.
Дата изменения

Этот диалог позволяет проводить поиск файлов, созданных или измененных в течение указанного промежутка времени.

Расширенный

Этот диалог позволяет искать файлы заданного типа, например, документы в формате Postscript, изображения в формате JPEG или звуки WAV.

3.3 Зона просмотра

Зона просмотра содержит результаты поиска. После выделения файла становятся доступными функции открытия, добавления в архив, удаления, просмотра свойств и открытия содержащего каталога. Двойной щелчок вызывает выполнение.
