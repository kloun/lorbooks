4. Меню

4.1 Файл

Из меню "Файл" доступны следующие функции:

Открыть

Этот пункт обращается к kfm для открытия выделенного файла, используя действие по умолчанию, например, программа будет выполнена, текст будет открыт в текстовом редакторе и т.д. Это действие можно выполнить и двойным щелчком мыши по имени файла.

Добавить в архив

Добавляет выделенный файл к архиву.

Удалить файл

Удаляет выделенный файл.

Свойства

Позволяет изменить свойства выделенного файла.

Открыть папку, в которой содержится файл

Открывает каталог, содержащий выделенный файл, в окне менеджера файлов(KFM).

Сохранить поиск

Сохраняет список выделенных файлов.

Quit

Выход из Kfind

4.2 Редактировать

Подпункты меню неактивны в текущей версии.

4.3 Настройки

Из этого пункта меню вы открываете диалог Preferences (Установки). Первая закладка диалога позволяет выбрать имя и формат файла, в котором хранятся результаты поиска. Вторая закладка позволяет менять установки архивации.

Сохранение

Укажите имя файла, в котором сохраняются результаты поиска. По умолчанию это .kfind-results.html в домашнем каталоге пользователя. Если используется другое имя, можно указать формат файла. В настоящее время допустимы форматы HTML и текстовый (.txt) .

Архиваторы

Этот диалог позволяет просматривать (а в будущем, возможно, и редактировать) определения архиваторов, используемые функцией Добавить в архив.
