4. Звуковые события

KWM включает несколько звуковых событий, каждое из которых может быть использовано в звуковом системном модуле. На данный момент задействованы следующие звуковые события:

  • Window Iconify
  • Window DeIconify
  • Window Maximize
  • Window UnMaximize
  • Window Sticky
  • Window UnSticky
  • Window New
  • Window Delete
  • Window New Trans
  • Window Delete Trans
  • Window Shade Up
  • Window Shade Down
  • Logout
  • Logout Message
