3. Настройка панели

Kpanel является очень гибкой и легко настраиваемой частью пользовательского интерфейса KDE. Эта секция содержит инструкции, которые помогут вам подстроить панель согласно вашему вкусу. Для информации о файле kpanelrc, файле настройки панели, обратитесь к секции Руководство для разработчиков.

Включенные ниже инструкции рассказывают как настраивать

  • Расположение и поведение панели
  • Кнопки панели
  • Виртуальные рабочие столы
  • Диспетчер приложений

3.1 Расположение и поведение панели

Для настройки расположения и поведения как Панели так и Панели задач, щелкните правой кнопкой мыши на панель в зоне без пиктограмм и выберите в меню пункт "Настройка". Щелкните кнопку Помощь для получения детальной информации об установках.

3.2 Кнопки панели

Вы можете передвигать, убирать или изменять функции кнопок панели, щелкая правой кнопкой мыши и выбирая желаемый параметр из появляющегося меню.

Для того, чтобы добавить кнопку на панель используйте один из следующих трех методов:

  • Для того, чтобы поместить на панель кнопку списка окон, выберите Диспетчер приложений > Панель > Добавить список окон.

  • Чтобы разместить программу из Диспетчера приложений на панель, выберите Диспетчер приложений > Панель > Добавить приложение, затем следуйте меню для выбора желаемой программы. Вы, также, можете расположить подменю из Диспетчера приложений прямо на панель, используя аналогичную процедуру; когда вы попадете в желаемое подменю, выберите первую позицию в подменю, которое является указателем на это не него. Например, для того, чтобы поместить подменю "System" на панель, выберите Диспетчер приложений > Панель > Добавить приложение > System > System.

  • Перетащите пиктограмму приложения из окна kfm или рабочего стола на панель. На панели не могут быть размещены только файлы .kdelnk и папки, содержащие файлы .kdelnk. Как только вы разместили пиктограмму на панели, щелкните по ней правой кнопкой мыши и выберите из меню пункт "Свойства" для того, чтобы поменять в них установки. Примечание: когда вы разместите пиктограмму на панели последним методом, будьте внимательны -- не удаляйте исходный файл или папку .kdelnk, иначе панель не сможет показывать эту пиктограмму.

Ярлыки-подсказки: kpanel показывает ярлыки для кнопок на панели и для строк в списке Диспетчера приложений. Для того, чтобы изменить ярлык кнопки, щелкните правой кнопкой мыши и выберите "Свойства" из меню. Щелкните на "Приложение" и наберите желаемый текст для ярлыка в поле "Комментарий". Для того, чтобы изменение ярлыка действительно произошло, вам, может быть, придется перезапустить kpanel (щелкните правой кнопкой на панель и выберите "Перезагрузить").

Поглощение программы: В разделе "Выполнить", в команде "Свойства" для кнопки программы, есть установки "Пиктограмма на панели", которые позволяют иметь кнопку на панели, показывающую работающую программу вместо статической пиктограммы. Это особенно полезно для маленьких программ, таких, например, как kbiff, которые служат для маленьких сообщений или показывают статус чего-либо. Вот как это делается:

  • Введите в графе "Выполнять" имя программы, которая должна будет запускаться на панели. Заметьте, что это графа "Выполнять" в разделе "Пиктограмма на панели".
  • Установите в "Заголовок окна" тот же заголовок, какой появляется в заголовке окна программы, когда она запускается "нормальным" образом на рабочем столе. Kpanel использует этот заголовок для идентификации окна, которое должно передвинуться на панель. Если это окно не имеет заголовка, запустите данную программу с рабочего стола, посмотрите заголовок на панели задач и используйте этот заголовок.

3.3 Виртуальные рабочие столы

Для того, чтобы поменять наименования виртуальных рабочих столов на панели, или для изменения числа рабочих столов, щелкните правой кнопкой мыши по свободной от кнопок части панели и выберите "Настройка" из меню. Щелкните на раздел "Рабочие столы" в окне "Конфигурирование КPanel" и произведите желаемые изменения (щелкните "Помощь" для более подробной информации об установках).

3.4 Диспетчер приложений

Для того, чтобы настроить зону пользователя в Диспетчере приложений, выберите Диспетчер приложений> Панель > Редактировать меню. Следуйте инструкциям в утилите kmenuedit для ввода желаемых изменений.

Чтобы ввести новые строки в первичном меню, щелкните правой кнопкой на панель в зоне без пиктограмм, и выберите в меню пункт "Настройка". Щелкните раздел "Настройки" и выберите "Персональные вначале".
