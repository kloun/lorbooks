мЮЯРПНИЙЮ ЯЕРЕБШУ ЯЙПХОРНБ

щРЮ ВЮЯРЭ МСФМЮ ДКЪ МЮЯРПНИЙХ ЯЕРЕБНИ ЙЮПРШ.

еЯКХ С БЮЯ МЕР ЯЕРЕБНИ ЙЮПРШ, БЮЛ МЕ РПЕАСЕРЯЪ ЯНГДЮБЮРЭ ХКХ МЮЯРПЮХБЮРЭ ТЮИКШ, НРМНЯЪЫХЕЯЪ Й ЯЕРЕБНИ ЙЮПРЕ. б ЩРНЛ ЯКСВЮЕ СДЮКХРЕ ЯХЛКХМЙ network ХГ БЯЕУ ДХПЕЙРНПХИ (/etc/rc.d/rc*.d)

мЮЯРПНИЙЮ ЬКЧГЮ ОН СЛНКВЮМХЧ.

еЯКХ БЮЬ ЙНЛОЭЧРЕП ЪБКЪЕРЯЪ ВЮЯРЭЧ ЯЕРХ БЮЛ ОНРПЕАСЕРЯЪ СЙЮГЮРЭ ЬКЧГ ХЯОНКЭГСЕЛШС ОН СЛНКВЮМХЧ ДКЪ БЮЬЕЦН ЙНЛОЭЧРЕПЮ. щРН ДЕКЮЕРЯЪ ДНАaБКЕМХЕЛ ЯКЕДСЧЫХУ ГМЮВЕМХИ Б ТЮИК /etc/sysconfig/network. гЮОСЯРХРЕ РЮЙСЧ ЙНЛЛЮМДС:

cat >> /etc/sysconfig/network << "EOF"
GATEWAY=192.168.1.2
GATEWAY_IF=eth0
EOF

гМЮВЕМХЪ ДКЪ GATEWAY Х GATEWAY_IF МЮДН ХГЛЕМХРЭ МЮ СЯРЮМНБЙХ БЮЬЕИ ЯЕРХ. GATEWAY ДНКФЕМХ ЯНДЕПФЮРЭ IP ЮДПЕЯ ЬКЧГЮ ОН СЛЮКВЮМХЧ, Х GATEWAY_IF СЙЮГШБЮРЭ МЮ ХМРЕПТЕИЯ ЯЕРЕБНИ ЙЮПРШ ВЕПЕГ ЙНРНПШИ ХЯОНЭГНБЮРЭ ДНЯРСО Й ЬКЧГС ОН СЛНКВЮМХЧ.

яНГДЮМХЕ ЙНМТХЦСПЮЖХНММШУ ТЮИКНБ ДКЪ ЯЕРЕБНИ ЙЮПРШ

йЮЙНИ ЯЕРЕБНИ ХМРЕПТЕИЯ АСДЕР ГЮОСЯЙЮРЭЯЪ Х НЯРЮМЮБКХБЮРЭЯЪ ЯЕРЕБШЛХ ЯЙПХОРЮЛХ ГЮБХЯХР НР ТЮИКНБ ПЮЯОНКНФЕММШУ Б ДХПЕЙРНПХХ /etc/sysconfig/network-devices. щРЮ ДХПЕЙРНПХЪ ДНКФМЮ ЯНДЕПФЮРЭ ТЮИКШ Б ТНПЛЮРЕ ifconfig.xyz, ЦДЕ xyz- ХЛЪ ЯЕРЕБНЦН ХМРЕПТЕИЯЮ (МЮОПХЛЕП eth0 ХКХ eth0:1).

еЯКХ БШ ПЕЬХКХ ОЕПЕХЛЕМНБЮРЭ ХКХ ОЕПЕЛЕЯРХРЭ ДХПЕЙРНПХЧ /etc/sysconfig/network-devices , САЕДХРЕЯЭ, ВРН БШ ОНОПЮБХКХ ТЮИКШ Б /etc/sysconfig/rc МЮ ОПЕДЛЕР МНБШУ ОСРЕИ Й ЯЕРЕБШЛ СЯРПНИЯРБЮЛ Б ЯЙПХОРЮУ.

рЕОЕПЭ, МНБШЕ ТЮИКШ ЯНГДЮММШЕ Б ДХПЕЙРНПХХ /etc/sysconfig/network-devices ЯНДЕПФЮР ЯКЕДСЧЫЕЕ. щРНИ ЙНЛЮМДНИ ЯНГДЮДХЛ ОПХЛЕП ТЮИКЮ ifconfig.eth0 Now, new files are created in that directory containing the following. : 

cat > /etc/sysconfig/network-devices/ifconfig.eth0 << "EOF"
ONBOOT=yes
IP=192.168.1.1
NETMASK=255.255.255.0
BROADCAST=192.168.1.255
EOF

еЯРЕЯРБЕММН, ГМЮВЕМХЪ ЩРХУ ОЕПЕЛЕММШУ ДНКФМШ АШРЭ ХГЛЕМЕМШ Б ЙЮФДНЛ ТЮИКЕ МЮ ОПЮБХКЭМШЕ МЮЯРПНИЙХ. еБЯКХ ОЕПЕЛЕММЮЪ ONBOOT СЯРЮМНБКЕМЮ yes, РН ЯЙПХОР БШОНКМХРЯЪ ОПХ ГЮЦПСГЙЕ ЯХЯРЕЛШ. еЯКХ СЯРЮМНБКЕМН ВРН МХРЭ НРКХВМНЕ НР yes, РН ЯЙПХОР МЕ БШОНКМХРЯЪ.
