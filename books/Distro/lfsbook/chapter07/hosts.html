CНГДЮМХЕ ТЮИКЮ /etc/hosts

еЯКХ ЯЕРЕБЮЪ ЙЮПРЮ ЯЙНМТХЦСПХПНБЮМЮ, РН БШ ДНКФМШ НОПЕДЕКХРЭ IP ЮДПЕЯ ДКЪ ЯХЯРЕЛШ Х БНГЛНФМШЕ ЮКХЮЯШ (ДНОНКМХРЕКЭМШЕ ХЛЕМЮ) ДКЪ ХЯОНКЭГНБЮМХЪ Б ТЮИКЕ /etc/hosts. яХМРЮЙЯ ЯКЕДСЧЫХИ:

<IP address> myhost.mydomain.org aliases

сАЕДХРЕЯЭ, ВРН ББЕДЕММШИ IP ЮДПЕЯ ЯХЯРЕЛШ ЪБКЪЕРЯЪ ВЮЯРЭЧ ОПЮБХКЭМНЦН ЮДПЕЯМНЦН ОПНЯРПЮМЯРБЮ, РХОЮ: 

    Class Networks
    A     10.0.0.0
    B     172.16.0.0 through 172.31.0.0
    C     192.168.0.0 through 192.168.255.0

оПЮБХКЭМШИ IP ЮДПЕЯ ЛНФЕР АШРЭ 192.168.1.1. оПЮБХКЭМНЕ ОНКМНЕ ХЛЪ ЯХЯРЕЛШ ДКЪ ЩРНЦН IP ЛНФЕР АШРЭ www.linuxfromscratch.org.

еЯКХ БШ МЕ УНРХРЕ ХЛОНКЭГНБЮРЭ ЯЕРЕБСЧ ЙЮПРС, РН БШ РЮЙХ ДНКФМШ ББЕЯРХ ОНКМНЕ ХЛЪ ЯХЯРЕЛШ (FQDN)ДКЪ ОПЮБХКЭМНИ ПЮАНРШ МЕЙНРНПШУ ОПНЦПЮЛЛ.

еЯКХ С БЮЯ МЕ ОКЮМХПСЕРЯЪ ХЯОНКЭГНБЮМХЕ ЯЕРЕБНИ ЙЮПРШ, ЯНГДЮИРЕ ЯКЕДСЧЫХИ ТЮИК: /etc/hosts ГЮОСЯРХБ ЙНЛЮМДС:

cat > /etc/hosts << "EOF"
# Begin /etc/hosts (no network card version)

127.0.0.1 www.mydomain.com <value of HOSTNAME> localhost

# End /etc/hosts (no network card version)
EOF

еЯКХ С БЮЯ ЕЯРЭ ЯЕРЕБЮЪ ЙЮПРЮ Х НМЮ МЮЯРПНЕМЮ, ХКХ БШ ОКЮМХПСЕРЕ ХЯОНКЭГНБЮРЭ ЯЕРЕБСЧ ЙЮПРС, РН ЯНГДЮИРЕ ЯКЕДСЧЫХИ ТЮИК /etc/hostsХЯОНКЭГСЪ РЮЙСЧ ЙНЛЮМДС:

cat > /etc/hosts << "EOF"
# Begin /etc/hosts (network card version)

127.0.0.1 localhost.localdomain localhost
192.168.1.1 www.mydomain.org <value of HOSTNAME>

# End /etc/hosts (network card version)
EOF

еЯРЕЯРБЕММН, ЮДПЕЯ 192.168.1.1 Х www.mydomain.org МЮДН ГЮЛЕМХРЭ МЮ ЮДПЕЯ Х ХЛЪ, ЙНРНПНЕ БЮЛ АНКЭЬЕ МПЮБХРЯЪ (ХКХ РПЕАСЕРЯЪ ББЕЯРХ ЮДПЕЯ Х ХЛЪ, ЙНРНПНЕ ОНОПНЯХК ББЕЯРХ ЯЕРЕБНИ/ЯХЯРЕЛМШИ ЮДЛХМХЯРПЮРНП Х ОКЮМХПСЕРЯЪ, ВРН ЩРЮ ЛЮЬХМЮ АСДЕР ВЮЯРЭЧ ЯСЫЕЯРБСЧЫЕИ ЯЕРХ.
