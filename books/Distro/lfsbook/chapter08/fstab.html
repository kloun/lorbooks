мЮГЮДцКЮБЮ 8. дЕКЮЕЛ ЯХЯРЕЛС LFS ГЮЦПСГНВМНИ.бОЕПЕД

яНГДЮМХЕ ТЮИКЮ /etc/fstab

яХЯРЕЛЮ ДНКФМЮ ГМЮРЭ Н ОЮПРХЖХЪУ, ЙНРНПШЕ ЕЯРЭ Б ЯХЯРЕЛЕ Х Н ОНПЪДЙЕ ХУ ЛНМРХПНБЮМХЪ. бЯЕ ОЮПРХЖХХ Х ОНПЪДНЙ ХУ ЛНРХПНБЮМХЪ ОЕПЕВХЯКЪЕРЯЪ Б ТЮИКЕ /etc/fstab. яНГДЮДХЛ РЮЙНИ ТЮИК /etc/fstab ЯНДЕПФЮЫХИ ЯКЕДСЧЫЕЕ:

cat > /etc/fstab << "EOF"
# Begin /etc/fstab

# filesystem   mount-point fs-type    options     dump    fsck-order

/dev/*LFS*     /           *fs-type*  defaults    1       1
/dev/*swap*    swap        swap       pri=1       0       0
proc           /proc       proc       defaults    0       0

# End /etc/fstab
EOF

*LFS*, *swap* and *fs-type* ДНКФМШ АШРЭ ГЮЛЕМЕМШ ОПЮБХКЭМШЛХ ГМЮВЕМХЪЛХ (МЮОПХЛЕП /dev/hda2, /dev/hda5 Х reiserfs).

йНЦДЮ ДНАЮБКЪЕРЯЪ ОЮПРХЖХЪ reiserfs, РН ГМЮВЕМХЕ 1 1 Б ЙНМЖЕ ЯРПНЙХ ДНКФМН АШРЭ ГЮЛЕМЕМН МЮ 0 0.

дКЪ АНКЕЕ ДЕРЮКЭМНИ ХМТНПЛЮЖХХ ГМЮВЕМХЪ ОНКЕИ Б ТЮИКЕ fstab ЯЛНРПХРЕ man 5 fstab.

пЮЯЯЛНРПХЛ ДПСЦХЕ ЯРПНЙХ, ЙНРНПШЕ, БНГЛНФМН, БЮЛ ОПХДЕРЯЪ ДНАЮБХРЭ Б БЮЬ ТЮИК fstab . дКЪ ОПХЛЕПЮ ПЮЯЯЛНРПХЛ БНГЛНФМНЯРЭ БЙКЧВЕМХЪ devpts:

devpts         /dev/pts    devpts     gid=4,mode=620 0    0

дПСЦНИ ОПХЛЕП, ЯРПНЙЮ, ОНГБНКЪЧЫЮЪ ХЯОНКЭГНБЮРЭ СЯРПНИЯРБЮ USB :

usbdevfs       /proc/bus/usb usbdevfs defaults    0       0

нАЕ НОЖХХ АСДСР ПЮАНРЮРЭ РНКЭЙН Б ЯКСВЮЕ, ЕЯКХ БШ БЙНЛОХКХПНБЮКХ Б ЪДПН БНГЛНФМНЯРЭ ОНДЙКЧВЕМХЪ ДЮММШУ СЯРПНИЯРБ.
