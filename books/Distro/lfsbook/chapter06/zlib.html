Установка Zlib-1.1.4

Приблизительное время компиляции:           0.07 SBU
Необходимое дисковое пространство:  1 MB

Инсталляция Zlib

Для инсталляции Zlib выполните:

./configure --prefix=/usr --shared &&
make LIBS="libz.so.1.1.4 libz.a" &&
make LIBS="libz.so.1.1.4 libz.a" install &&
mv /usr/lib/libz.so.* /lib &&
ln -sf ../../lib/libz.so.1 /usr/lib/libz.so &&
cp zlib.3 /usr/share/man/man3

Содержимое Zlib

Последняя проверка: версия 1.1.4.
