Конфигурация

Основным конфигурационным файлом является, естественно, ~/.vimrc. Активация русского шрифта в GUI-режиме, плюс выбор темы для обоих режимов осуществляется, например, следующим кодом: 

if has("gui_running")
    colorscheme candy
    set guifont=-cronyx-courier-medium-r-normal-*-*-120-*-*-m-*-koi8-r
endif

if !has("gui_running")
    colorscheme elflord
endif

Быстрые и, вероятно, привычные клавиши: 

"Выход по F10
nmap <F10> :qall<CR>
imap <F10> <ESC>:qall<CR>
"Сохранение по F2
nmap <F2> :w<CR>
imap <F2> <ESC>:w<CR>i<Right>
"Компиляция по F9
nmap <F9> :make<CR>
imap <F9> <ESC>:make<CR>

Конечно же в ViMприсутствует подробная документация по настройкам -- :options. Также рекомендую ознакомиться с файлом vimrc.forall, автор которого постарался пояснить каждый, могущий оказаться вам полезным параметр настройки ViM...

Также вы можете ознакомиться с моим файлом .vimrc.
