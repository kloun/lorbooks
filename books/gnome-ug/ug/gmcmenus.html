Меню диспетчера файлов

В этом разделе подробно описаны все команды, включенные в меню диспетчера файлов.

Подсказка: Все меню диспетчера файлов GNOME - "отрывные": если выбрать в меню полоску перфорации (узкая полоска в самом верху меню), то меню "оторвется" от окна диспетчера файлов и превратится в отдельное окно, которое вы можете поместить в любое место экрана.

  • Файл - это меню содержит команды, связанные с файлами и команды обшего назначения.

    • Новое окно - эта команда откроет новое окно диспетчера файлов.

    • Открыть - эта команда откроет ранее выбранный вами файл.

    • Копировать - эта команда запустит окно-диалог Копия, с помощью которого вы сможете скопировать выбранный файл(ы).

    • Переместить - эта команда запустит окно-диалог Переместить, с помощью которого вы можете переместить или переименовать выбранный файл(ы).

    • Удалить - эта команда уничтожит выбранный файл(ы).

    • Закрыть окно - эта команда закроет окно диспетчера файлов.

  • Правка - это меню содержит команды, связанные с редактированием (правкой) и с выбором файлов.

    • Выделить все - эта команда выберет все файлы в текущем каталоге.

    • Выделить файлы - эта команда запустит окно-диалог, в котором вы можете ввести критерий для отбора файлов. Например, чтобы выбрать все файлы, начинающиеся с буквы D, введите D* и нажмите OK.

    • Инвертировать выделение - эта команда обратит текущий выбор файлов. Например, если ваш каталог содержит 10 файлов, из которых вы выбрали 7, то команда Инвертировать выделение отменит выбор этих семи файлов и наоборот, выберет оставшиеся три.

    • Пересмотреть каталог - эта команда пересмотрит текущий каталог на тот случай, если какие-то файлы были изменены вне диспетчера файлов.

    • Настройка - эта команда запустит диалог настройки, с помощью которого вы можете изменить настройку диспетчера файлов GNOME.

  • Макет - это меню содержит команды, с помощью которых можно управлять внешним видом и расположением окон диспетчера файлов.

    • Сортировать по - эта команда запустит окно-диалог, в котором вы можете указать, в каком порядке следует показывать файлы в текущем каталоге. Файлы могут быть упорядочены в соответствии с одним из следующих признаков:

      • Имя

      • Тип файла

      • Размер

      • Время последнего доступа

      • Время последней модификации

      • Время последнего изменения

    • Избиратеьный - эта команда запустит окно-диалог, с помощью которого вы можете настроить диспетчер файлов так, чтобы он показывал только файлы, отвечающие некоторому критерию. Например, если вы хотите, чтобы он показывал только файлы, начинающиеся на букву D, наберите в этом окне D* и нажмите OK. Если вы захотите снова видеть все файлы, вызовите этот диалог еще раз и сотрите ранее введенный критерий.

    • Icons - переключить диспетчер файлов в режим "пиктограммы": каждый файл будет представлен большой пиктограммой.

    • Brief View - переключить диспетчер файлов в краткий режим, когда он показывает только имена файлов.

    • Detailed View - переключить диспетчер файлов в подробный режим, когда он показывает всю информацию про каждый файл.

    • Custom View - переключить переключит диспетчер файлов в специальный режим, в котором вы можете выбрать, какую именно информацию про файл следует показывать. Вы можете настроить специальный режим с помощью раздела Специальный в диалоге Настройки (см. Разд. Настройка диспетчера файлов..

  • Команды - в этом меню содержатся команды, которые обрабатывают файлы в диспетчере файлов GNOME.

    • Найти файл - эта команда вызывает окно-диалог, с помощью которого вы можете найти файлы на вашей системе, отвечающие заданному критерию.

    • Edit mime types - эта команда запустит капплет редактирования MIME типов. Этот капплет позволяет вам задать приложения, используемые для открывания, просмотра и редактирования файлов разных типов. Подробное описание этого капплета дано в Разд. GNOME типы Mime в Гл. 9

    • Исполнить - эта команда позволяет вам запустить команду операционной системы из GMC.

    • Исполнить на панели - This menu item lets you run preloaded commands or commands you specify within the directory that you are currently in. Such commands might be to find SUID or SGID programs, etc.

    • Выход - с помощью этой команды вы можете выйти из диспетчера файлов.

      ВАЖНАЯ ИНФОРМАЦИЯ: Команда Выход остановит все процессы, запущенные диспетчером файлов, в том числе процесс, обеспечивающий функционирование рабочего стола GNOME. В связи с этим, использование этой команды не рекомендуется.
