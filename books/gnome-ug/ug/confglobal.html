Глобальные свойства панели.

Чтобы изменить настройку панели GNOME, щелкните правой кнопкой мыши на панель и выберите в меню команду Глобальная настройка. Вы также можете вызвать эту команду с помощью дочернего меню Панель в главном меню GNOME.

На экране появится диалог глобальной настройки, с помощью которого вы можете установить параметры для всех используемых вами панелей. Все панели, которые вы создадите впоследствии, также будут использовать эти параметры.

Рисунок 7-1. Диалог глобальной настройки

Диалог глобальной настройки состоит из шести разделов: Анимация, Кнопки запуска, Ящики, Меню, Кнопка завершения работы и Разное. Подробное описание каждого из этих разделов дается ниже.

Раздел "Анимация"

  • Разрешить анимацию - эта команда включает и выключает анимацию в GNOME. Все остальные параметры этого раздела имеют смысл и могут быть изменены только если анимация включена. По умолчанию, анимация включена.

  • Скорость анимации авто-сокрытия - если вы используете авто-сокрытие панели, то этот параметр задает скорость авто-сокрытия.

  • Скорость анимации ручного сокрытия - этот параметр задает скорость сокрытия панели при нажатии кнопки сокрытия.

  • Скорость анимации ящиков - этот параметр задает скорость выдвижения ящиков на панели.

  • Задержка авто-сокрытия (мсек) - если вы используете авто-сокрытие панели, то этот параметр задает время задержки (в миллисекундах): панель спрячется через указанное время после того, как курсор мыши сдвигается с панели. Чтобы восстановить панель, поместите курсор мыши на то место, где была панель.

  • Размер панели при авто-сокрытии (пикселы) - если вы используете авто-сокрытие панели, то этот параметр задает число пикселов, занимаемых панелью в спрятанном состоянии.

Раздел "Кнопка запуска"

  • Плитки в окнах - этот режим позволяет использовать "плитки", т. е. файлы с картинками, в качестве фона под кнопками запуска приложений.

  • Плитка на кнопке (не нажатой) - здесь вы можете выбрать файл с изображением, используемым в качестве фона под не нажатой кнопкой. Вы можете нажать кнопку Просмотр, чтобы найти требуемый файл на вашей системе. Этот параметр имеет смысл и может быть изменен, только если плитки разрешены.

  • Плитка на кнопке (нажатой) - здесь вы можете выбрать файл с изображением, используемым в качестве фона под нажатой кнопкой. Вы можете нажать кнопку Просмотр, чтобы найти требуемый файл на вашей системе. Этот параметр имеет смысл и может быть изменен, только если плитки разрешены.

  • Ширина каймы (плитка) - этот параметр задает ширину каймы вокруг пиктограммы. Это полезно в тех случаях, когда ваша пиктограмма полностью покрывает плитку: в этом случае вы можете уменьшить ширину каймы, так что часть плитки станет видима. Этот параметр имеет смысл и может быть изменен, только если плитки разрешены.

  • Глубина (смещение при нажатии) - этот параметр задает глубину смещения кнопки запуска при нажатии.

  • Плитка на кнопке (не нажатой) - здесь вы можете выбрать файл с изображением, используемым в качестве фона под не нажатой кнопкой ящика. Вы можете нажать кнопку Просмотр, чтобы найти требуемый файл на вашей системе. Этот параметр имеет смысл и может быть изменен, только если плитки разрешены.

  • Плитка на кнопке (нажатой) - здесь вы можете выбрать файл с изображением, используемым в качестве фона под нажатой кнопкой меню. Вы можете нажать кнопку Просмотр, чтобы найти требуемый файл на вашей системе. Этот параметр имеет смысл и может быть изменен, только если плитки разрешены.

Раздел "Кнопка завершения работы"

  • Кнопки в плитках - этот режим позволяет использовать "плитки" в качестве фона для кнопки завершения работы.

  • Плитка на кнопке (не нажатой) - здесь вы можете выбрать файл с изображением, используемым в качестве фона под не нажатой кнопкой завершения работы. Вы можете нажать кнопку Просмотр, чтобы найти требуемый файл на вашей системе. Этот параметр имеет смысл и может быть изменен, только если плитки разрешены.

  • Плитка на кнопке (нажатой) - здесь вы можете выбрать файл с изображением, используемым в качестве фона под нажатой кнопкой завершения работы. Вы можете нажать кнопку Просмотр, чтобы найти требуемый файл на вашей системе. Этот параметр имеет смысл и может быть изменен, только если плитки разрешены.

Раздел "Разное"

  • Подсказки включены - этот режим включает подсказки для кнопок и апплетов на панели. Подсказки - это короткие справочные сообщения, которые появляются, если вы задержите курсор мыши на одном из объектов на панели.

  • Показывать маленькие пиктограммы - в этом режиме элементы меню будут сопровождаться маленькими пиктограммами.

  • Показывать ... кнопки - этот режим включает показ трех точек рядом с теми командами в меню, которые запускают диалоги.

  • Показывать всплывающие меню вне панели - в этом режиме всплывающие меню всегда помещаются вне панели. Если его отключить, то они могут также быть помещены поверх панели - это бывает полезно при работе с низким разрешением экрана.

  • Сохранять меню в памяти - в этом режиме GNOME будет сохранять меню в памяти, вместо того, чтобы просматривать их каждый раз, когда вы нажимаете на кнопку меню. Это ускоряет работу, но при этом вы можете не увидеть в меню команды, добавленные после того, как вы выбрали этот режим.

  • Движение с перестановкой других апплетов - в этом режиме при движении или добавление апплета на панели он может двигать соседние апплеты (например, если он не помещается в промежуток между ними).

  • Свободное движение (не двигать другие апплеты) - этот режим фиксирует текущее расположение апплетов и других объектов на панели, так что если вы двигаете один из них, остальные остаются на своих местах - тем самым, вы можете передвинуть апплет только на свободное место. Этот режим удобна, если вам нравится текущее расположение апплетов на панели.

  • Спрашивать подтверждение - в этом режиме GNOME будет запрашивать подтверждение перед завершением работы.

  • Поднимать панель под мышью - если вы используете несовместимый с GNOME диспетчер окон, то может случиться, что окна приложений частично закрывают панель. В этом случае вы можете выбрать этот режим и панель будет "подниматься" над окнами приложений как только курсор мыши оказывается над панелью.

  • Оставлять панель под окнами - если ваш диспетчер окон совместим с GNOME, то он знает о существовании панели и по умолчанию располагает окна приложений так, чтобы они не закрывали панель. Вы можете разрешить ему помещать окна приложений поверх панели, выбрав этот режим. Это бывает полезно при работе с низким разрешением экрана.

  • Закрывать ящик после запуска - по умолчанию, ящик остается выдвинут после того, как вы нажали одну из кнопок запуска в нем; чтобы его закрыть, вы должны щелкнуть на него мышью. Если вы хотите, чтобы ящик закрывался сам после того, как вы нажали одну из кнопок запуска, выберите этот режим.

  • Отступ между апплетами - этот параметр задает минимальный интервал между апплетами и другими объектами на панели.
