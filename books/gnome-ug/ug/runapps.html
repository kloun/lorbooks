Запуск приложений

GNOME предоставляет несколько способов запустить приложение. Одно из достоинств GNOME состоит в том, что он предоставляет вам удобный графический интерфейс для запуска приложений и контроля за их работой. Ниже приводится список способов запуска приложений:

Как вы уже видели в предыдущем разделе, главное меню и его дочерние меню содержат большой список приложений. Чтобы запустить любое из них, достаточно выбрать соответствующую строчку в меню. Вы также можете добавить или убрать приложения из меню с помощью редактора меню, который обсуждается в Гл. 8.

Вы также можете запустить приложения с помощью кнопок запуска на панели. Добавление к панели кнопок запуска обсуждалось в Разд. Добавление кнопок запуска приложений.

Если вы пользуетесь диспетчером файлов GNOME, то вы можете запустить любой исполняемый файл двойным щелчком мыши.

Наконец, можно запустить любое приложение с помощью команды Run program из главного меню GNOME. Когда вы выбираете эту команду, появляется окно-диалог, в котором вы можете набрать команду для запуска приложения. Например, если вы хотите запустить редактор Emacs, но его нет среди приложений в ваших меню и на панели, то вы можете воспользоваться командой Run program и набрать команду emacs в появившемся диалоге.
