Запуск приложений из диспетчера файлов

Вы можете запустить приложения из диспетчера файлов GNOME просто двойным щелчком: если с данным типом файлов связано какое-то приложение, то оно будет запущено. Например, двойной щелчок на файл с расширением html запустит Netscape. Вы можете указать, какое именно приложение следует использовать для открывания файлов данного типа - см. Разд. GNOME типы Mime в Гл. 9

Если с данным типом файлов не связано никаких приложений, то вы можете щелкнуть на него правой кнопкой мыши и выбрать в появившемся меню Open with. На экране возникнет окно-диалог, в котором вы можете указать, какое приложение должно использоваться для открывания этого файла. Например, если вы хотите отредактировать файл test.txt с помощью популярного редактора Emacs, щелкните на этот файл правой кнопкой мыши, выберите Open with в появившемся меню, наберите emacs перед именем файла test.txt в окне-диалоге и щелкните на OK. Будет запущен редактор Emacs, который откроет файл test.txt.
