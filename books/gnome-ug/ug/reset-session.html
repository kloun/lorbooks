"Чистый сеанс"

Одна из продвинутых возможностей управления сеансами GNOME - это возможность вернуться к "чистому" сеансу работы GNOME, если с вашим сеансом что-то не в порядке. Для этого надо держать нажатыми клавиши CTRL и SHIFT в момент начала сеанса работы.

На экране возникнет окно-диалог, которое позволяет вам выбрать одну из двух возможностей.

Вы можете выбрать Start with default programs. Этот выбор удалит всю информацию о приложениях, работавших при завершении предыдущего сеанса GNOME. Он не изменит настройки GNOME; в частности, все сделанные вами изменения в капплете диспетчера сеансов останутся в силе.

Вторая возможность - это выбрать Reset all user settings. Это вернет настройку всех приложений и основных компонентов GNOME к стандартным значениям, тем самым уничтожив все сделанные вами изменения настройки панели GNOME, диспетчера файлов, диспетчера сеансов и других приложений. Этот выбор не уничтожит файлы на вашем рабочем столе.

ВАЖНАЯ ИНФОРМАЦИЯ: Обе эти возможности предназначены только для опытных пользователей и должны использоваться только в случае проблем с вашим сеансом работы. Их использование чревато возможностью потери данных для ваших приложений.
