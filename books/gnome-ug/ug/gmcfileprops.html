Свойства и операции с файлом

Если щелкнуть на любой файл в окне диспетчера файлов правой кнопкой мыши, то появится меню, с помощью которого можно изменить различные свойства этого файла или произвести другие операции с ним:

  • Открыть - эта команда откроет файл, используя приложение, связанное с данным типом файлов. (См. Разд. GNOME типы Mime в Гл. 9 по поводу того, как изменить эти связи.)

  • Open With - эта команда позволяет вам выбрать приложение, с помощью которого следует открыть данный файл. Подробное описание см. в Разд. Запуск приложений из диспетчера файлов.

  • Просмотр - эта команда покажет содержимое файла, используя простую встроенную программу просмотра текстовых файлов.

  • Правка - эта команда запустит редактор, связанный с данным типом файлов. (См. Разд. GNOME типы Mime в Гл. 9 по поводу того, как изменить эти связи.)

  • Копировать - эта команда скопирует файл в clipboard для последующей вставки.

  • Удалить - эта команда уничтожит файл.

  • Переместить - эта команда запустит диалог перемещения файла. Подробное описание этого диалога дано в Разд. Копирование и перемещение файлов.

  • Свойства - эта команда запустит диалог Свойства, с помощью которого вы можете посмотреть и изменить различные параметры выбранного файла.

    Диалог Свойства состоит из трех разделов: Статистика, Options, and Permissions.

    Рисунок 6-6. File Properties

    • Статистика - в этом разделе показана такая информация как имя, размер, тип и история файла. Вы можете изменить имя файла, набрав новое имя в графе "Имя".

    • Options - в этом разделе вы можете изменить действия, связанные с командами для открывания, просмотра и редактирования файла. Если команда для открывания файла должна запускаться в терминальном окне, выберите режим Needs terminal to run.

      Замечание: Если вы запустите диалог Свойства для одной из пиктограмм на рабочем столе, то вы также можете изменить используемую пиктограмму.

    • Permissions - в этом разделе вы можете изменить допуски и имя владельца файла - если, конечно, вы имеете на это право. Вы можете установить допуски на чтение, запись и запуск для владельца файла, группы и для всех остальных. Вы также можете сменить имя владельца файла и установить специальные свойства: UID, GID и Sticky (предупреждение: последнее предназначено только для опытных пользователей - не пробуйте это делать, если вы не вполне понимаете, что это значит!).
