Основы редактирования

Blender предполагает управление двумя руками. Для того чтобы уменьшить нагрузку на мышку практически все команды продублированы горячими клавишами на клавиатуре. Основные приведены в меню "Панель инструментов" (нажмите ПРОБЕЛ), из которого их можно вызвать. И, наконец, некоторые команды можно задавать мнемонически, мышкой рисуя на экране символы.

Команды ввода

С помощью мышки

По мере возможности мы постарались стандартизировать функции кнопок мышки следующим образом:

  • Левая : открывает ввод в кнопки, поля, двигает слайдеры и перемещает 3D-курсор.
  • Правая+CTRL: добавляет вершину.

  • Средняя: изменить вид или - во время редактирования - закрепляет внесенные изменения.
  • Средняя+SHIFT: панорамирование.
  • Средняя+CTRL: масштаб (zoom in/out).

  • Правая: выбирает Объект и делает его активным.
  • Правая+SHIFT: выбор нескольких Объектов.
  • Правая+CTRL: выбирает и делает активными тот Объект, к центру которого ближе 3D курсор.
  • Правая: перемещение (нажать-двигать-отпустить).


Через клавиатуру

В программе приняты следующие функциональные клавиши:

  • F1 - F12: глобальные горячие клавиши, функции которых не зависят от типа окна. Наиболее важные из них: F1: Загрузить файл Blender. F2: Сохранить файл Blender. F3: Сохранить изображение (image). F12: Рендеринг.
  • ESC: выход из любого меню или процесса (редактирование, рендеринг, выбор файла, и т.д.)
  • Цифровая клавиатура: горячие клавиши выбора вида.
  • Пробел: вызывает меню "Панель инструментов" со списком основных горячих клавиш.
