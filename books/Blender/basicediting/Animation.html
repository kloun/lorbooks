Основы редактирования

Анимация по ключевым кадрам и с помощью анимационных кривых

В программах 3D-анимации используют два метода анимации трехмерных Объектов:
  • С помощью ключевых кадров. Аниматор задает положение Объекта в определенные моменты времени (ключевые кадры), а программа интерполирует "промежуточные" (рассчитывает положение Объекта между ключевыми кадрами). Положение (состояние) Объекта в ключевых кадрах аниматор может менять.
  • Кривые движения (Motion curves). Для каждой оси XYZ можно построить кривые положения, угла поворота и размера. На графе движения по вертикали отложена зависимость значения кривой, а по горизонтали - время (номер кадра).
Оба этих метода объединены в Blender в одну систему "Ipo". Для создания и редактирования Ipo по клавише "I" в окне 3DWindow вызывается диалоговое окошко выбора предустановленных параметров.

Для блока Ipo надо существует специальное окно. Горячая клавиша - SHIFT+F6.

Принцип работы с блоком Ipo в Blender не зависит от того, что именно Вы собираетесь анимировать: движение Объекта или параметры Материала. Как только Вы освоите работу с Ipo Объекта, работа с другими Ipo становится самоочевидной.
